# NLSA_SAXS

This repository contains NLSA scripts and data.

## Overview

**NLSA_SAXS** is a collection of scripts and associated data files related to Nonlinear Laplacian Spectral Analysis (NLSA) and Small-Angle X-ray Scattering (SAXS). The repository is primarily written in Jupyter Notebook format.

## Features

- Jupyter Notebooks for NLSA analysis
- Example data for SAXS experiments
- Modular scripts for reproducibility

## Requirements

- Python (recommended: latest version)
- Jupyter Notebook
- Additional dependencies may be required based on the scripts (see notebook headers for details)

## Usage

1. Clone the repository:
    ```bash
    git clone https://github.com/UWM-CXFEL/NLSA_SAXS.git
    cd NLSA_SAXS
    ```

2. Open Jupyter Notebook:
    ```bash
    jupyter notebook
    ```

3. Explore and run the notebooks in your browser.
